## 권장 캡처 툴

- **정적 이미지**: Windows 스니핑 도구 (Win+Shift+S) 또는 ShareX
- **GIF**: [ScreenToGif](https://www.screentogif.com/) (경량, 무료)
- **해상도**: 1920x1080 권장 (PNG), GIF는 720p 이하로 압축

## 파일 크기 가이드

- PNG: 500KB 이하 (필요 시 [TinyPNG](https://tinypng.com/) 압축)
- GIF: 5MB 이하 (ScreenToGif 내장 최적화 사용)
